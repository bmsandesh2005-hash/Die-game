# Dice Game

A simple Python dice rolling game.

## Description
This program simulates rolling a dice.  
The user presses **Enter** to roll the dice and gets a random number between **1 and 6**.  
The game continues until the user chooses to quit.

## Features
- Random dice roll (1 to 6)
- Continuous play until exit
- Simple and user-friendly

## How to Run
1. Install Python 
2. Open terminal or command prompt
3. Run the program:
   
   python die_game.py
